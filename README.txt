Fikk det meste til å fungere, dog dokumentasjonen for polyline var vag på sitt beste
og fikk dermed ikke implementert den ordentlig. Ikke fungerende for øyeblikket.

GPS fungerer, samt markers til de forskjellige fasilitetene.
Markers måtte hardkodes, for det var ikke bare bare å legge til .csv filen som lå ute.
Konverterte .csv filen til en .kml fil som i prinsipp skulle være det Maps foretrekker. Uten hell.

For at dere skal kunne kjøre på deres mobiler må GPS være skrudd på, samt
jeg må legge til deres fotspor i API nøkkelen.

https://www.youtube.com/watch?v=cWDM0lZl_C0
Forklarer hvordan dere skal finne fotsporene som tilhører deres installasjon.
Når jeg har lagt til deres SHA1 fingerprint til API-nøkkelen skal den være kjørbar via USB.

Eventuelt lar det seg gjøre å vise den fra min enhet i lab.
